# RiverProject Wiki

Welcome to the RiverProject Wiki! This is a comprehensive guide to understanding and working with the River Project, a Year 2 project from Sheffield Hallam University.

## Project Overview

The River Project is an environmental IoT initiative focused on sensor data collection and analysis. It combines Raspberry Pi hardware with MQTT messaging, Node-RED data visualization, and Python data processing to create a complete environmental monitoring solution.

## Technologies Used

- **Python** - Main programming language
- **Raspberry Pi** - Hardware platform
- **Sense Hat** - Environmental sensor attachment
- **MQTT** - Message broker for real-time data
- **Node-RED** - Visual data dashboard and event logging
- **Data Logging** - Persistent sensor data storage

## Project Features

- Real-time environmental sensor data collection
- Rolling average calculations for data smoothing
- MQTT-based data publishing and subscription
- Node-RED dashboard for visualization
- Event logging system for field observations
- Comprehensive data analysis capabilities
